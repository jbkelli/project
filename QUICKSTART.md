## Running Your App

### Option 1: Using Expo Go (Easiest - No Emulator Needed)
1. Install "Expo Go" app on your phone:
   - iOS: Download from App Store
   - Android: Download from Play Store

2. Start the development server:
   ```
   npm start
   ```

3. Scan the QR code that appears with:
   - iOS: Use your Camera app
   - Android: Use the Expo Go app

### Option 2: Using Android Emulator
1. Make sure Android Studio is installed and an emulator is set up
2. Run:
   ```
   npm run android
   ```

### Option 3: Using iOS Simulator (Mac only)
1. Make sure Xcode is installed
2. Run:
   ```
   npm run ios
   ```

### Option 4: Run in Web Browser
```
npm run web
```
